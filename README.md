# Jupiter Kpodjadan - Interactive Portfolio

A modern, interactive portfolio website showcasing my experience in AI/ML engineering, research, and distributed systems.

## Features

- **Modern Design**: Sleek, gradient-based dark theme with smooth animations
- **Interactive Elements**: Smooth scrolling, hover effects, and scroll-reveal animations
- **Responsive**: Fully mobile-responsive design with adaptive layouts
- **Performance**: Optimized assets, lazy loading, and efficient CSS
- **Accessibility**: WCAG compliant with keyboard navigation support
- **SEO Friendly**: Proper semantic HTML and meta tags

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About**: Overview with key statistics
3. **Experience**: Timeline view of professional roles and research projects
4. **Projects**: Featured projects with technology stacks
5. **Education**: Academic background and certifications
6. **Skills**: Comprehensive skill categories with visual organization
7. **Awards**: Recognition and honors
8. **Contact**: Multiple ways to get in touch

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Design**: Responsive Grid/Flexbox, CSS Animations, Gradients
- **Icons**: Font Awesome 6
- **Performance**: Intersection Observer API, Lazy Loading
- **Hosting**: GitHub Pages

## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/JupiterSeho/portfolio.git
cd portfolio
```

2. Open `index.html` in your web browser, or use a local server:
```bash
python -m http.server 8000
# or
npm install -g http-server
http-server
```

3. Navigate to `http://localhost:8000` in your browser

### File Structure

```
portfolio/
├── index.html
├── css/
│   └── styles.css

```

## Customization

### Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    --accent-color: #8b5cf6;
    /* ... other colors ... */
}
```

### Content
Edit the HTML in `index.html` to update:
- Personal information
- Work experience
- Projects
- Skills
- Education

### Animations
Adjust animation timing and effects in `css/styles.css` or add new animations in `js/script.js`

## Deployment on GitHub Pages

1. Create a new repository named `username.github.io` (replace `username` with your GitHub username)
2. Push this portfolio to that repository
3. Your site will be live at `https://username.github.io`

Or, for a project site:
1. Push to any repository in your GitHub account
2. Go to Settings → Pages
3. Select the main branch as the source
4. Your site will be live at `https://username.github.io/repository-name`
